The problem

Intracellular current-clamp recordings from one neuron at 10 kHz. On each 0.5 s trial a fluctuating current was injected and the voltage recorded. The four files are the same cell driven at different current bandwidths (25 / 100 / 200 / 400 Hz); each trial is an independent episode. The cell rests near −60 mV and fires stereotyped action potentials at 25–42 Hz. The task: discover one equation, fit it, and predict the voltage for held-out current inputs.

The equation

A leaky membrane driven by I(t), with a spike-triggered adaptation current, a spike-triggered (moving) threshold, and a reset.

# subthreshold membrane
C·dV/dt = −gL(V − EL) + I(t) − Iadapt(t)
Iadapt(t) = Σk wk·zk(t)   # each spike adds 1 to every zk; zk decays with τk

# spike when V reaches a threshold that rises after each spike
spike if V(t) ≥ VT(t),   VT(t) = VT0 + aθ·θ(t)   # θ decays with τθ

# then reset
V → Vr,   held for tref

How the form came from the data

I did not assume the model. Three model-free views selected it:

1Dynamic I–V curve. Below ≈−50 mV the net ionic current I−C·dV/dt is a straight line through EL — a pure linear leak — then turns sharply inward above VT0 (the regenerative Na⁺ spike current).
2Phase plot. A sharp spike-initiation "elbow" and a stereotyped AP loop → a threshold + reset, not a continuous nonlinearity.
3Spike-triggered average. A reset followed by a slow after-hyperpolarisation → a spike-triggered adaptation current and a moving threshold.

Passive parameters were consistent across all four bandwidths — one cell, one equation.

Spike parameters were fit by minimising the Victor–Purpura distance (q = 4 s⁻¹) between model and data, pooled across conditions; the membrane and adaptation current came from a closed-form regression of dV/dt. Spikes were detected with an elbow (2nd-derivative) initiation-point method, applied identically to measured and predicted traces.

Where it fails

  • 100 Hz is the weakest condition — the model slightly over-fires (41→45 Hz) under sustained depolarisation with a fixed reset.
  • Exact spike timing can't be recovered below ~1–2 ms: near threshold a small voltage error shifts a spike by several ms, and every trial uses a different current (no repeat-reliability ceiling).
  • Startup transient — each held-out trial is initialised at EL (no pre-stimulus history), so the first ~20–30 ms can relax.
